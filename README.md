# 🌍 Celebrity Language Tutor

**Learn languages through immersive conversations with AI celebrity roleplay** — completely free, no login required.

## ✨ Features

- 🗣️ **7 Languages**: Spanish, French, Japanese, Korean, German, Italian, Chinese
- 🎭 **28 Celebrity Teachers**: Learn from Shakira, Jackie Chan, Bruce Lee, Miyazaki, and more
- 💬 **3 Interaction Modes**: Text Chat, Voice Mode, Video-Call UI
- 📚 **Adaptive Learning**: Beginner, Intermediate, and Advanced levels
- 🎯 **4 Scenarios**: Casual Chat, Travel, Interview, Cultural Immersion
- 🆓 **100% Free**: No ads, no subscription, no catch

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/celebrity-language-tutor.git
cd celebrity-language-tutor

# Install dependencies
bun install

# Run development server
bun run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 💻 Tech Stack

- **Framework**: Next.js 16 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **State**: Zustand with persistence
- **Animations**: Framer Motion
- **AI**: z-ai-web-dev-sdk (Chat + TTS)

## 💬 Interaction Modes

- **Text Mode**: Typing animations, follow-up messages, suggested replies
- **Voice Mode**: TTS audio responses, natural conversation flow
- **Video-Call UI**: Full-screen simulated video call with emotion reactions
