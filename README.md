# Celebrity Language Tutor

Learn languages with AI celebrity tutors! Practice Spanish, French, Japanese, Korean, German, Italian, and Chinese with AI personalities.

## Features

- 🗣️ 7 Languages supported
- 👤 28 Celebrity AI tutors
- 💬 Text, Voice, and Video call modes
- 📚 Interactive vocabulary building
- 🎯 Multiple learning scenarios

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:3000

## Deployment

This project is optimized for Vercel deployment.
